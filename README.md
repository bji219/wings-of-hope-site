# Wings of Hope

**Rise. Heal. Connect.**

A sanctuary website for women with endometriosis, providing community support, resources, and hope.

## Deployment

This site is configured for Netlify deployment.

### Setup Instructions

1. Connect this repository to your Netlify account
2. The site will automatically deploy from the branch
3. No build process is required - this is a static HTML site

### Configuration

- `index.html` - Main website file
- `netlify.toml` - Netlify configuration with security headers and redirects

## Important Note

**Missing Asset:** The site references a logo image file `Wings_of_Hope_1.png` that needs to be added to the repository root directory.

Please add your logo file before deploying to ensure the site displays correctly.

## Features

- Responsive design with mobile optimization
- Animated gradient backgrounds and floating elements
- Smooth scroll animations
- Community resources and support sections
- Stories of hope from real users

## Technologies

- Pure HTML/CSS/JavaScript (no build tools required)
- Google Fonts: Cormorant Garamond & Outfit
- CSS animations and transitions
- Intersection Observer API for scroll animations
